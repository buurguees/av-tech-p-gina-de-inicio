# Guía de clasificación del catálogo NEXO AV

Criterio operativo para clasificar productos y servicios en el catálogo. Consultar antes de crear categorías nuevas o dudar entre varias.

## PA (PRODUCTOS AUDIOVISUALES)

| Código | Nombre | Criterio |
|--------|--------|----------|
| PA-01 | PANTALLAS LED | Pantallas LED, módulos, videowalls |
| PA-02 | CARTELERÍA DIGITAL | Displays de cartelería, totems, quioscos |
| PA-03 | ILUMINACIÓN PROFESIONAL | Luminarias, proyectores, control de luz |
| PA-04 | CONTROL, PROCESADO Y CONECTIVIDAD AV | Controladores, procesadores, switches de señal, patch panels de señal AV |
| PA-05 | AUDIO PROFESIONAL | Altavoces, mezcladoras, micrófonos |
| PA-06 | RACKS Y ACCESORIOS | Rack físico, armarios rack, cabinas, flight cases, accesorios de rack (ventilación, bandejas, etc.) |
| PA-07 | DISPLAYS LCD | Monitores y displays LCD |
| PA-08 | ACCESORIOS | Accesorios genéricos AV no cubiertos por otras subcategorías |

## MI (MATERIAL DE INSTALACIÓN)

Material de instalación y estructura de montaje: cableado, canalización, perfiles, soportes, tornillería, consumibles.

| Código | Nombre | Criterio |
|--------|--------|----------|
| MI-01 | TORNILLERÍA Y FIJACIONES | Tornillos, anclajes, fijaciones |
| MI-02 | CABLEADO | Cables de señal, alimentación, datos |
| MI-03 | CONSUMIBLES | Cintas, bridas, etiquetas |
| MI-04 | MATERIAL ELÉCTRICO | Regletas, bases, conectores |
| MI-05 | SOPORTES Y ANCLAJES | Soporte de pantallas, anclajes de pared |
| MI-06 | CANALIZACIÓN Y PROTECCIÓN | Canaletas, tubos, protecciones |
| MI-07 | PERFILES Y ESTRUCTURAS | Perfiles metálicos, estructuras de montaje |

## Regla rápida

- **Rack físico, armario, cabina, flight case, accesorios de rack** → PA-06
- **Switch de señal, patch panel de señal, controlador, procesador AV** → PA-04
- **Cable, canaleta, perfil, tornillo, consumible de obra** → MI
