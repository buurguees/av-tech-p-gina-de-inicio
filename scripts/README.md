# Scripts

## Criterio de organizacion

Los scripts deben agruparse por dominio operativo y no dejarse sueltos en raiz.

Convencion objetivo:

- `scripts/sharepoint/`: sincronizacion y archivado documental
- `scripts/marketing/`: tareas de la web publica
- `scripts/nexo/`: utilidades de NEXO AV
- `scripts/shared/`: scripts reutilizables por varios dominios
